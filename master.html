Standort erforderlich

Diese App funktioniert nur mit aktiviertem Standort. Ohne GPS-Freigabe kann die App nicht verwendet werden.

Der Standort wird für die korrekte Funktionsweise der App benötigt.

GPS ist blockiert! So behebst du es:

1. Schließe den Inkognito-Modus und öffne die Seite im normalen Browser

2. Tippe auf das 🔒 Schloss neben der URL

3. Gehe zu Berechtigungen → Standort → Erlauben

4. Seite neu laden

Wichtig: GPS/Standort muss auch in den Handy-Einstellungen aktiviert sein!

📍 GPS ist im Browser blockiert

Dein Browser blockiert den Standortzugriff. So aktivierst du ihn:

Chrome (Handy):

1. Tippe auf das 🔒 Schloss links neben der URL

2. Tippe auf "Berechtigungen"

3. Setze "Standort" auf Erlauben

4. Seite neu laden

Chrome (Desktop):

1. Klicke auf das 🔒 Schloss in der Adressleiste

2. Bei "Standort" → "Erlauben" wählen

3. Seite neu laden (F5)

Safari (iPhone):

1. Öffne Einstellungen → Safari → Standort

2. Wähle "Erlauben"

3. Seite neu laden

Wichtig: Auch das Handy/PC muss GPS/Standort in den Geräte-Einstellungen aktiviert haben!

📲 App installieren

Empfohlen: Google Chrome

1. Öffne diese Seite in Chrome

2. Tippe auf ⋮ (oben rechts)

3. Wähle "App installieren"

Funktioniert dann auch offline!

Safari (iPhone):

1. Tippe auf □↑ (Teilen, unten Mitte)

2. Wähle "Zum Home-Bildschirm"
